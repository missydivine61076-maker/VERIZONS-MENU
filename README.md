# VERIZONS-MENU
**A powerful mod menu suite for multiple games**

## Features

### Gorilla Tag Mod Menu
- 🔊 Soundboard with custom sounds
- 👻 Ghost Monkey mode
- 👹 Ghost Trolls mode
- 🤏 Pull Mod with 3% speed boost
- 🔫 Ban Gun (30-minute bans)

### Rivals Mod Menu
- 🎨 Cosmetics & Skins
- 🔊 Soundboard with victory/defeat sounds
- 📊 Game Stats Display
- ⚙️ Settings & Customization
- ✨ UI Enhancements & Trail Effects

## Installation

### Rivals (Roblox)
1. Use a Roblox executor (Synapse X, Script-Ware, etc.)
2. Copy the code from `RivalsModMenu.lua`
3. Paste and execute in the game

### Gorilla Tag (VR)
1. Ensure GorillaTag is installed and running
2. Open your mod loader (MelonLoader/BepInEx)
3. Copy code from `GorillaTagMods.lua`
4. Load the mod in your executor
5. Press **K** to open the mod menu in-game

## Controls
- **Insert Key** - Toggle menu visibility
- **Click buttons** - Activate features
